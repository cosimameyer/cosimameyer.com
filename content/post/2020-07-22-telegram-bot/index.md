What would you do if you had to learn several hundred flashcards? A great way is to have a bot that automatically sends you these cards and constantly reminds you about the things that you have to learn 🤓

[Dennis Hammerschmidt](https://dennis-hammerschmidt.netlify.com/) and I wrote a bot with Python and AWS Lambda. It's just a few line of codes -- if you want to write one yourself, you're lucky: we have a short overview with the [necessary steps](https://github.com/dennis-hammerschmidt/telegram-bot) that you have to follow.
